---
title: SetCheckpoint
---

Set a checkpoint at the specified coordinates with the given size.

**Parameters:**

- **fX**: X coordinate of the checkpoint as float
- **fY**: Y coordinate of the checkpoint as float
- **fZ**: Z coordinate of the checkpoint as float
- **fSize**: Size of the checkpoint as float

**Returns:** `true` on success, `false` on failure

**Signature:**

```ahk
SetCheckpoint(fX, fY, fZ, fSize)
```
