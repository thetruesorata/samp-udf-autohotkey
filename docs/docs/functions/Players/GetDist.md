---
title: GetDist
---

Calculate the Euclidean distance between two 3D positions.

**Parameters:**

- **pos1**: First position array `[x,y,z]`
- **pos2**: Second position array `[x,y,z]`

**Returns:** Distance as float, or `0` if inputs invalid

**Signature:**

```autohotkey
GetDist(pos1, pos2)
```
