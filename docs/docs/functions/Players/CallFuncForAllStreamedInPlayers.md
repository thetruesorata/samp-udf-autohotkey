---
title: CallFuncForAllStreamedInPlayers
---

Call a callback function for every streamed-in player.

**Parameters:**

- **cfunc**: Function name to call for each player (receives the player object)
- **dist**: Optional max distance filter (default: large)

**Returns:** `true` on success, `false` otherwise

**Signature:**

```autohotkey
CallFuncForAllStreamedInPlayers(cfunc, dist=0x7fffffff)
```
