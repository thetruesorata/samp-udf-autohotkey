---
title: IsPlayerInRangeOfPoint2D
---

Check whether the player is within a certain range of a point in 2D space.

**Parameters:**

- **_posX**: X coordinate of the point as float
- **_posY**: Y coordinate of the point as float
- **_posRadius**: Radius around the point as float

**Returns:** `true` if the player is within range, `false` otherwise

**Signature:**

```autohotkey
IsPlayerInRangeOfPoint2D(_posX, _posY, _posRadius)
```
