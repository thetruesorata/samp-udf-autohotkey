---
title: IsPlayerInRangeOfPoint
---

Check whether the player is within a certain range of a point in 3D space.

**Parameters:**

- **_posX**: X coordinate of the point as float
- **_posY**: Y coordinate of the point as float
- **_posZ**: Z coordinate of the point as float
- **_posRadius**: Radius around the point as float

**Returns:** `true` if the player is within range, `false` otherwise

**Signature:**

```autohotkey
IsPlayerInRangeOfPoint(_posX, _posY, _posZ, _posRadius)
```
